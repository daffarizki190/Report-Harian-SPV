# 🚀 Laravel 11 Modern Tricks & Best Practices
*Gandaria City Project Hub Modernization Summary*

Ini adalah rangkuman teknik "Senior Developer" yang kita terapkan untuk mengubah project monolitik biasa menjadi aplikasi modern berperforma tinggi.

---

### 1. Database & Eloquent Optimization
*   **`latestOfMany()`**: Gunakan ini untuk mendapatkan relasi terbaru (misal: Laporan Terakhir) sebagai relasi *One-to-One*. Sangat efisien dan mencegah N+1 Query.
    ```php
    public function latestReport() {
        return $this->hasOne(Report::class)->latestOfMany();
    }
    ```
*   **Eager Loading with API Resources**: Selalu gunakan `with()` di controller dan bungkus hasilnya dengan `JsonResource`. Gunakan `whenLoaded()` di dalam resource agar relasi hanya muncul jika dipanggil.

### 2. Backend Architecture & Clean Code
*   **Pipeline Pattern**: Pindahkan logika filter query yang berantakan di controller ke dalam class-class filter kecil yang diproses lewat `Pipeline`. Controller jadi super bersih!
*   **Form Requests & `prepareForValidation()`**: Jangan sanitasi data di Controller. Gunakan method ini di FormRequest untuk *trim*, *lowercase*, atau *slugging* data sebelum validasi berjalan.
    ```php
    protected function prepareForValidation() {
        $this->merge(['username' => strtolower($this->username)]);
    }
    ```
*   **Laravel 11 `whereAny()`**: Pengganti `orWhere` yang elegan untuk pencarian di banyak kolom sekaligus.
    ```php
    $query->whereAny(['name', 'email', 'phone'], 'LIKE', "%$search%");
    ```

### 3. Performance & API Speed
*   **Laravel 11 `defer()`**: Fitur paling sakti untuk memindahkan proses berat (logging, email, notifikasi) ke *background* tanpa perlu setup Redis/Queue. Response API jadi instan!
    ```php
    defer(fn() => Log::info('Aktivitas dicatat'));
    ```
*   **Laravel 11 `once()`**: Memoization tingkat dewa. Hasil fungsi akan disimpan di RAM selama siklus request tersebut. Cocok untuk query berat yang dipanggil berkali-kali.
    ```php
    $stats = once(fn() => Report::count());
    ```

### 4. Code Quality & Security
*   **Pest Architecture Testing**: Pasang "polisi kode" otomatis. Pastikan tidak ada `dd()`, `dump()`, atau pemanggilan `env()` langsung di Controller yang bisa merusak performa atau keamanan.
    ```php
    test('no globals')->expect(['dd', 'dump'])->not->toBeUsed();
    ```
*   **Temporary Signed URLs**: Gunakan untuk fitur seperti "Magic Link" agar login aman tanpa password dan tanpa perlu tabel token tambahan di database.

### 5. Frontend Strategy (Modern Monolith)
*   **Persistent Layout**: Desain layout yang tidak render ulang saat pindah menu (Sidebar & Header tetap di tempat). Memberikan sensasi SPA (Single Page Application) meskipun pakai Blade biasa.
*   **Debounce Search**: Implementasikan *delay* (misal 500ms) pada input pencarian di JavaScript untuk menekan jumlah request ke server.
